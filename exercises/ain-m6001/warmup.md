# AIN-M6001 — Warm-up

Complete before or alongside notebook `01_first_marketing_world.ipynb`.

```{exercise}
:label: ex-m6001-ontology

List **at least five** entities you would include in a *personal marketing world model* (not necessarily the same as the notebook v1). For each entity, name **one property** you would measure and **one** you are tempted to ignore because it is hard to quantify.
```

```{exercise}
:label: ex-m6001-tick

Define your **clock**: what is one “tick” in your simulation (hour, day, week)? What changes *within* a tick vs *between* ticks?
```

```{solution} ex-m6001-ontology
:label: sol-m6001-ontology

There is no single correct list. Strong answers distinguish **stocks** (credibility, backlog) from **flows** (posts per week) and name at least one **hidden** variable (e.g. recovery, shame, platform risk) they are postponing.
```

```{solution} ex-m6001-tick
:label: sol-m6001-tick

A **day** is a common first tick; intraday ticks are justified if modeling attention blocks explicitly. Answers should admit what the model **aggregates away** inside a tick.
```
