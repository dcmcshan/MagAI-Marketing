# Exercises

*Authoritative copy: part of the MyST project (`myst.yml`).*

Structured practice uses MyST **`{exercise}`** / **`{solution}`** directives ([MyST exercises](https://mystmd.org/guide/exercises)). Per-course files live under `exercises/<course-code>/`; notebooks may reference the same labels for gated cells.

Add patterns or files to `project.toc` in `myst.yml` under **Exercises**.
