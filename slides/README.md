# Slides (instructor & classroom)

*Authoritative copy: part of the MyST project (`myst.yml`).*

Slide decks live here as **MyST Markdown**. Separators `+++` delimit **frames** for Beamer-style export (see [MyST Beamer / slides](https://mystmd.org/jtex/create-a-beamer-template)); the same files are included in the **Jupyter Book** build for web (`jupyter-book build --site`).

Add new decks to `project.toc` in `myst.yml` under **Slides**.
